The object of MASTERMIND is to guess a secret code consisting of a series of 4 colored pegs. Each gues results in a feedback (white for a peg with the right color at the wrong location and black for a correctly placed and colored peg) narrowing down the possibilities of the hidden code.

For this slightly altered version of the game, 2 players will have to play simultaneously, and the first one to guess the secret code will be declared winner of the game.

Good luck !
